# 产品需求文档

## 场景

把模糊的想法整理成别人能理解并行动的沟通成果。

## 需求

以用户故事地图的格式展示：

| 用户活动 | 用户任务 | 用户细节 |
|----------|----------|----------|
| 对话（Dialogue） | 开始对话 | • 输入消息到聊天框<br>• 查看 AI 回复 |
| 对话（Dialogue） | 采纳便签 | • 查看便签建议卡片<br>• 点击采纳 |
| 白板（Whiteboard） | 挑选便签 | • 浏览便签列表<br>• 勾选需要的便签 |
| 白板（Whiteboard） | 生成备忘 | • 点击"生成备忘录"按钮<br>• 自动跳转至画布 |
| 画布（Canvas） | 编辑备忘 | • 查看自动生成草稿<br>• 选择收件人<br>• 选择目的 |
| 画布（Canvas） | 发送结果 | • 点击"发送"按钮<br>• 显示成功提示（Toast） |
