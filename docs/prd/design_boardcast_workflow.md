# 设计备忘广播流程

## 用户故事

作为一个管理者，我在大范围传递消息并且希望看到各级回复时，我需要进行详细的广播流程设计，以确保消息能够有效传达并得到预期的反馈。

## 典型案例

### 启动

原始消息："给商务发备忘，让她去招生工作组收集想法，结论发给COO"

### 加工

AI提出问题：
1. 主题是什么
2. 沟通意图是什么
3. 如何跟踪反馈

人整理回答：
1. COO：接手CEO的教练角色，需要告诉主管和商务如何分工并建立汇报关系
2. COO助理：要熟悉如何配合COO组织流程
3. 新主管：熟悉自己的管理角色，与项目组成员建立汇报关系。
4. 新商务：熟悉自己的角色，进入团队工作节奏中。

### 传递

- 发件人：CEO 办公室（CEO、CEO助理）
- 收件人：COO 办公室（COO、COO助理）、招生工作组（主管、商务）
